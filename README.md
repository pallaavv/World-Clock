🕒 WorldClock
A simple clock website that mainly shows the current IST (India Standard Time) and also lets you check the time in different parts of the world. I built this project to practice my web development skills. The design is basic, and I focused more on making the time display work correctly.

📦 Technologies

HTML

CSS

JavaScript

🦄 Features
Here’s what you can do with WorldClock:

View IST: Always see the current India Standard Time.

World Time: Check the current time in different countries.

Simple UI: Easy to read and understand.

👩🏽‍🍳 The Process
I started by creating a digital clock that shows IST. Then I added functionality to display world times by using JavaScript’s Date object and time zones. I kept the design simple so I could focus on learning how to work with time and dates in code.

📚 What I Learned

How to use JavaScript’s Date object.

How to handle different time zones.

How to update the clock in real time using setInterval.

Basics of styling with CSS.

💭 How can it be improved?

Add more countries and cities.

Add dark mode and light mode.

Show the date along with the time.
